#### 项目说明

##### 厦门大学本科生绩点计算工具

敝校的那个破烂教务系统里只有科目的成绩，没有绩点，每次都得手动计算麻烦至极，就写了这个小工具，输入教务系统的账号密码后会打印所有主修科目的成绩、绩点，各个学期的绩点以及全部学期的总绩点。

没什么技术含量，练习一下Python的基础语法和简单的爬虫技巧罢了，倒是每次用自己的成绩去测试代码，低得可怜的绩点总是刺伤本学渣的内心。
